Flower-Pot
Flower Pot is a beginner HTML & CSS project focused on learning core CSS concepts like the box model, selectors, pseudo-classes, and Flexbox. It features a navbar, sidebar, and responsive card layout to practice spacing, styling, and hover effects without JavaScript.
